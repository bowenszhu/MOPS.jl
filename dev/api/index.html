MOPS.arm — Method
arm(κ::AbstractVector, i::Integer, j::Integer) -> Any

Compute the arm-length of a partition at a square.

Given the diagram of the partition κ, the arm-length of the (i, j) square S is the number of squares to the right of S, in other words arm(κ, i, j) = κ[j] - i.

Examples

┌───┬───┬───┬───┐
│   │ S │   │   │
├───┼───┼───┼───┘
│   │   │   │
├───┼───┴───┘
│   │
├───┤
│   │
└───┘
julia> arm([4, 3, 1, 1], 2, 1)
2

MOPS.gsfact — Method
gsfact(α, r, κ::AbstractVector) -> Any

Compute the generalized shifted factorial.

The generalized shifted factorial extends the definition of factorials to include partition parameters. More precisely

\[\operatorname{gsfact}(α,r,κ)= ∏_{i=1}^{\operatorname{length}(κ)}\operatorname{sfact}\left(r-\frac{i-1}{α},κ_i\right).\]

See also: sfact.

Examples

julia> gsfact(1, 3, [3, 2, 1]) |> Int
360
source

MOPS.lhook — Method
lhook(α, κ::AbstractVector, i::Integer, j::Integer) -> Any

Compute the lower hook length of a partition at a square.

Given the parameter α, the partition κ and the location (i, j),

\[\operatorname{lhook}(α,κ,i,j)=\operatorname{leg}(κ,i,j)+1+α\operatorname{arm}(κ,i,j)\]

See also: arm, leg, uhook.

Examples

│   │   │
├───┼───┘
│   │
└───┘
julia> lhook(2, [4, 2, 1], 2, 1)
6

MOPS.parvalid — Method
parvalid(κ::AbstractVector) -> Bool

Check if the given partition is valid.

A partition $\kappa$ is a finite, ordered, non-increasing sequence of positive integers $\kappa_1\geq\kappa_1\geq\kappa_1\geq\ldots\geq\kappa_l$.

Examples

julia> parvalid(Int[])
true

julia> parvalid([4, 3, 1, 1])
true

julia> parvalid([5, 3, 4])
false

MOPS.rho — Method
rho(α, κ::AbstractVector) -> Any

The $ρ$ function of a partition.

Given the Jack parameter (real positive integer) α and the partition κ,

\[ρ(α,κ)=∑_{i=1}^{\operatorname{length}(κ)}κ_i\left(κ_i-1-\frac{2(i-1)}{α}\right)\]

Examples

julia> rho(2, [3, 1]) |> Int
5
source
MOPS.sfact — Method
sfact(r, k) -> Any

Compute the shifted factorial, which is the ratio of two factorials

\[\frac{(r+k+1)!}{(r-1)!}\]

for integral k or equivalently the ratio of two Gamma functions

\[\frac{Γ(r+k)}{Γ(r)}.\]

The shifted factorial is also known as the Pochhammer symbol and rising factorial.

Examples

julia> sfact(3, 5)
2520

MOPS.uhook — Method
uhook(α, κ::AbstractVector, i::Integer, j::Integer) -> Any

Compute the upper hook length of a partition at a square.

Given the parameter α, the partition κ and the location (i, j),

\[\operatorname{uhook}(α,κ,i,j)=\operatorname{leg}(κ,i,j)+α(1+\operatorname{arm}(κ,i,j))\]
